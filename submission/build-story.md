# Build Story

CareerTwin OS was built as a different category from resume builders, job boards, or interview tools. The core idea is that career planning should behave like a simulation system: state, constraints, alternate futures, assumptions, shocks, evidence, decisions, and replay.

The app starts with a fictional demo profile so reviewers can see the complete product without signup. The deterministic engine remains useful when credentials are absent, while Netlify Functions prepare AI and official market-data enrichment for configured production environments.
