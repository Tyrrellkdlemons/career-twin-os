# Architecture

CareerTwin OS uses a four-layer architecture:

1. Structured occupation, skill, evidence, and constraint data.
2. Deterministic scoring, route planning, shock recalculation, and seeded uncertainty.
3. Interactive React routes for simulation, comparison, evidence, missions, reflection, and replay.
4. Optional Netlify Functions for OpenAI, O*NET, and BLS enrichment.

The system prioritizes traceability and honest uncertainty over fake prediction precision.
